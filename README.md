# Projeto Integrador

## Apresentação

Bem-vindo ao repositório do Projeto Integrador desenvolvido pelo grupo.
Este espaço será utilizado para organizar, desenvolver e documentar todas as etapas do projeto ao longo do semestre.

O objetivo deste repositório é centralizar o código, os documentos e os materiais relacionados ao desenvolvimento da solução proposta pelo grupo.

---

## Status do Projeto

Atualmente o grupo encontra-se na fase inicial de planejamento.
A ideia do software que será desenvolvido ainda está em processo de definição.

As próximas etapas incluem:

* definição do problema a ser resolvido
* escolha da proposta de software
* levantamento de requisitos
* definição das tecnologias a serem utilizadas
* planejamento do desenvolvimento

---

## Estrutura do Repositório

O projeto está organizado nas seguintes pastas:

```
backend/
frontend/
docs/
```

**backend**
Contém a implementação da lógica de negócio, APIs e integração com banco de dados.

**frontend**
Contém a interface do usuário e toda a parte visual da aplicação.

**docs**
Destinada à documentação do projeto, incluindo requisitos, diagramas, relatórios e materiais de apoio.

---

## Fluxo de Trabalho

Para manter a organização do projeto, o desenvolvimento seguirá um fluxo baseado em branches.

* A branch `main` conterá a versão estável do projeto.
* Novas funcionalidades devem ser desenvolvidas em branches separadas.
* Alterações devem ser integradas por meio de Pull Requests.
