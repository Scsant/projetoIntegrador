Objetivo
Refinar o design de um sistema SaaS educacional chamado EscolaAgenda, focado em gestão de agenda escolar automatizada.

Contexto
O sistema já possui as seguintes páginas:

Landing Page
Login
Recuperação de senha
Seleção de perfil
Dashboard principal
Agenda escolar (calendário mensal, semanal e diário)
Agenda do professor
Painel da direção
Centro de notificações

O sistema utiliza um design minimalista institucional com tons neutros e azul como cor primária.

Tecnologias do produto
React + TypeScript
Tailwind CSS
Framer Motion
Lucide Icons

Requisitos de UI/UX

Melhorar o design para nível SaaS profissional.
