Objetivo
Criar a interface principal de agenda escolar inteligente.

Contexto
A agenda escolar possui eventos com:

- data
- período
- atividade
- detalhes
- participantes
- recursos

Exemplos reais:
reunião pedagógica
avaliação diagnóstica
reunião de pais
planejamento escolar
tutoria

Interface

Visualização calendário

Modo:
- mensal
- semanal
- diário

Eventos aparecem como cards com:
- título
- horário
- participantes
- categoria

Filtros
- professor
- turma
- tipo de atividade
- direção

Funcionalidades

Criar evento
Editar evento
Enviar notificação
Adicionar participantes

Design
- calendário moderno
- navegação clara
- cores neutras
- ícones institucionais

Entregáveis
- calendário completo
- visualização semanal
- criação de evento
